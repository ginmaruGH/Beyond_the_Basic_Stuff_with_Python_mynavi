## 仮想環境の構築

```bash
python3 -m venv bbsp_env
```

```bash
# 仮想環境への切り替え
source bbsp_env/bin/activate
```
source 
```bash
# pipのアップグレード
pip install --upgrade pip
```

```bash
# 仮想環境にインストールされているパッケージの一覧作成
pip freeze > requirements.txt
```

```bash
# 仮想環境の終了
deactivate
```
